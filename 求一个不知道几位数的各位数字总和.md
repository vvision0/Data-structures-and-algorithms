求一个不知道几位数的各位数字总和

```c#
    public int AddDigits(int num) {
        while(num >= 10)
        {
            int temp = 0;
            while(num > 0)
            {
                temp += num % 10;
                num /= 10;
            }
            num = temp;
        }
        return num;
    }
```
